# USGBC-Rebuilding Guide

An interactive post-fire rebuilding guide aligned with USGBC/green-resilience principles for the Altadena community recovery effort.

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Mapping**: Leaflet.js for interactive maps
- **3D Visualization**: Cesium ion Story Viewer
- **3D Models**: Google `<model-viewer>` for ADU exploration
- **Styling**: Modern CSS reset, custom responsive design
- **Accessibility**: WCAG compliant with keyboard navigation and screen reader support

## Usage
1. Open `index.html` in a modern web browser
2. Navigate sections using the sidebar menu
3. Expand/collapse content panels for detailed information
4. Use interactive tools to explore rebuilding options
5. Access the 3D Story Viewer for immersive geospatial visualization

## Navigation Tips for 3D Story Viewer
- **Left click + drag** to rotate the view
- **Right click + drag** to zoom in/out  
- **Middle click + drag** to pan
- **Touch gestures** supported on mobile devices
- **Fullscreen mode** available for expanded viewing
